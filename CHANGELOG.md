# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2024-11-12

### Added
- Initial release of bentogrid library
- `computeBentoLayout` function for generating bento grid layouts
- Two layout algorithms: squarified and binary
- TypeScript type definitions
- Support for ESM and CommonJS formats
- Configurable canvas dimensions, padding, and gutters
- Proportional tile area system
- Comprehensive test suite
- Demo application with interactive controls

### Features
- Framework-agnostic design (works with React, Vue, vanilla JS, etc.)
- Zero runtime dependencies
- Optimized squarified treemap algorithm for square-like tiles
- Binary split algorithm for deterministic layouts
- Full TypeScript support with declaration files
